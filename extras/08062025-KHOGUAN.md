## uncomment some pre-installed lua files

AstroNvim template 予咱所有佇 $config/ 下底个檔案，干焦 init.lua 佮
lua/lazy_setup.lua 是開起來个，其他个攏用 `if true then return {} end`
佇檔頭就關起來矣。

1. lua/community.lua : 伊有 lua pack，試看扑開了後，會自動安裝
   lua_ls 袂？A: 會！lua_ls / stylua / selene 三種攏起來！官網說明：
   - Adds lua Treesitter parser
   - Adds lua_ls language server
   - Adds stylua formatter
   - Adds selene linter
     - On aarch64 machines this is skipped.

2. lua pack 裝好了後，來試閣會出現 "undefined global `vim`" 袂？
   哈！袂矣啦！早知就攏用原裝个就好，欲改嘛愛確定知影家己改啥。

## UI 修改

1. 改 which-key 設定：popup 改用 helix 直式風格。
2. statusline mode 用文字顯示，無愛干焦用色水分別 mode。
3. bufferline 用 <Tab>/<S-Tab> 切換 buffer。

4. 伊有設 tabstop=2，但是 softtabstop 佮 shiftwidth 攏是 0，怪。
   補充：`~/.local/share/nvim-as1/lazy/AstroNvim/lua/astronvim/plugins/_astrocore_options.lua`
   愛好好仔了解一下，可能只要設 `tabstop` 佮 `expandtab` 就有夠額矣。

5. auto-completion for MD 真討厭，愛改！  
   我想講只要佇 lua/astrocore.lua 遐設 `cmp = false` 就好，到時任何 ft 欲用只要扑
   <C-Space> 就會將 blink auto-completion 叫出來，結果試袂出來。

   2025-08-09 補充：設做 `cmp = fasle` 了後，無表示無載入，只要用 `<leader>uc`
   即時就會使叫出來用，這完全符合我个需要啦！

```lua
      maps.n["<Leader>uc"] =
         { function() require("astrocore.toggles").buffer_cmp() end, desc = "Toggle autocompletion (buffer)" }
      maps.n["<Leader>uC"] =
         { function() require("astrocore.toggles").cmp() end, desc = "Toggle autocompletion (global)" }
```
