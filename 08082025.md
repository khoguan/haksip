## nvim-surround vs mini.surround

前者是延續早期 vim-surround 个操作方式，預設个 keymaps `ys`, `ds`, `cs`
佮 vim 本底个基本操作無相犯；後者較創新，用 `sa`, `sd`, `sr` 等，功能也較濟，
為著相沖个緣故，AstroNvim 另外共伊 map 做用 gz 做 prefix，閣加上 a / c / d 等。
flash.nvim 也是將 s remap 起來，真害！

nvim-surround 簡單用就好，免創甲遐複雜。

## bamboo theme

選用 `multiplex` style 是偏綠色个，袂䆀。

改色！comment 个色水預設傷光，bamboo 个文件真貼心，教儂按怎改：

```lua
require('bamboo').setup {
  colors = {
    bright_orange = '#ff8800', -- define a new color
    green = '#00ffaa', -- redefine an existing color
  },
  highlights = {
    -- make comments blend nicely with background, similar to other color schemes
    ['@comment'] = { fg = '$grey' },

    ['@keyword'] = { fg = '$green' },
    ['@string'] = { fg = '$bright_orange', bg = '#00ff00', fmt = 'bold' },
    ['@function'] = { fg = '#0000ff', sp = '$cyan', fmt = 'underline,italic' },
    ['@function.builtin'] = { fg = '#0059ff' },
  },
}
```

`$grey` (#818781) 傷殕，我改做 $light_grey 有較好淡薄，閣進一步改較光，
目前用 #879387，RGB 三色差不多，G 較濟小可，配合綠色底。

bamboo theme 个 palette 佇遮：
`~/.local/share/nvim-as1/lazy/bamboo.nvim/lua/bamboo/palette.lua`
有三套，對應三種風格 vulgaris, multiplex, light。
