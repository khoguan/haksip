## 二次函數由簡到繁

一、頂點佇原點，二次函數就是 $y = ax^2$，a 可能是正是負。

二、頂點佇 y 軸上下平移量 n（免管正負），新函數就是 $y = ax^2 + n$。

補充：假使題目共咱講一个二次函數圖形通過 (m, 0) 佮 (-m, 0) 兩點，按呢意思就是以
y 軸做對稱軸，函數圖形个頂點也就是 y 軸頂个某一个非原點，所以函數就是
$y = ax^2 + n$。

用一般式 $y = ax^2 + bx + c$ 來看，即種函數个一次項係數 b 就是 0。題目若閣共咱講
圖形通過某一个其他个點 (p, q)，按呢用 (m, 0) 佮 (p, q) 兩組已知數代入
$y = ax^2 + n$，聯立了後就算會出來完整个函數式。

例題：若二次函數 $y = ax^2 + bx + c$ 與 x 軸交點為 (6, 0), (-6, 0)，且過
(4, -4)，則 a, b, c = ?

解方：因為佮 x 軸个兩交點對稱於 y 軸，所以函數就是 $y = ax^2 + c$，代入 (6, 0)
佮 (4, -4)，得：

$$
\begin{cases}
0 &= 36a + c \quad\dots\text{(1)} \\
-4 &= 16a + c \quad\dots\text{(2)} \\
\end{cases}
\begin{aligned}
(1) - (2) \Rightarrow 4 = 20a \\
\therefore\ a = \frac{1}{5},\ c = -\frac{36}{5}
\end{aligned}
$$

註：若進前無想過，直接將兩點 (6, 0) 佮 (-6, 0) 代入一般式內去聯立，嘛會發現
b = 0 个事實。

三、若閣有左右平移，bx 項就出現矣！

四、假使 $y = ax^2$ 干焦左右平移 m，按呢就會變做 $y = a(x - m)^2$，若以配方式
來看，只有「二次項」；展開做一般式才有一次項佮常數項。

即種二次函數个頂點是 (m, 0)，頂點式个通式是 $y = a(x - h)^2 + k$（h, k) 是頂點，
(m, 0) 代入得著 $y = a(x - m)^2$ 仝款結果。

## Github Markdown 檔內數學式分行符號 `\\`

Markdown LaTeX/MathJax tip:  Github 頁面个 MD math block 內底愛閣包
`\begin{...}...\end{...}`，斷行符號 `\\` 才有效。而且內底最後一行 行尾个 `\\`
無效，著愛另起一个新个 math block 才會分行。`\begin{...}` 大括號內底會使得用
`aligned`、`cases`，其他猶未試。
