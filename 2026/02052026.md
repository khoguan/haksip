## 直線佮 $x,\ y$ 軸所夾个面積

ax + by + c = 0 个 x 截距是 -c/a；y 截距是 -c/b，所以該直線佮兩軸所夾个面積就是
|(-c/a)(-c/b)| / 2 = c² / (2|ab|)，用 MathJax 顯示如下：

$$
{ \left| \frac{-c}{a}\cdot\frac{-c}{b} \right| \over 2 } = { c^2 \over 2|ab| }
$$
