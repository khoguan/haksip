## 二次方程式 ax² + bx + c = 0 兩根个和佮積

### 戇直算法

$$\begin{aligned}
ax^2 + bx + c = 0 \\
\implies x = \frac{-b\pm\sqrt{b^2-4ac}}{2a} \\
\text{兩根个和：}\frac{-b\times 2}{2a} = \frac{-b}{a} \\
\text{兩根个積：}\frac{(-b)^2 - (\sqrt{b^2-4ac})^2}{4a^2} = \frac{c}{a}
\end{aligned}$$

### 反推法

令兩根是 $\alpha$ 佮 $\beta$，按呢：

$$\begin{aligned}
x = \alpha \lor x = \beta \\
x - \alpha = 0 \lor x - \beta = 0 \\
(x-\alpha)(x-\beta) = 0 \\
x^2 - (\alpha+\beta)x + \alpha\beta = 0 \quad\dots\text{(1)}
\end{aligned}$$

原方程式左右各除以 a，則 $x^2 + \frac{b}{a}x + \frac{c}{a} = 0$，比較 (1) 式可知：

$$\begin{cases}
\alpha + \beta = -\dfrac{b}{a} \\
\alpha \times \beta = \dfrac{c}{a}
\end{cases}$$
