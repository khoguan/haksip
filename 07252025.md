### Check the plugin status

```
:lua print(vim.inspect(require('ccc').setup))
```

### Check if the function exists in Neovim

```
:lua print(type(require('ccc').picker.custom_entries))
```

### Inspect the ccc module structure

```
:lua vim.print(require('ccc').picker)
```
This will show you all available functions in the ccc.picker module.

### Try to call the function directly

```
:lua print(pcall(function() return require('ccc').picker.custom_entries end))
```
This returns true if the function exists and can be called, false otherwise.

### Check the loaded modules

```nvim
:lua for k,v in pairs(package.loaded) do if k:match('ccc') then print(k) end end
```
This shows all loaded modules related to "ccc".

### Debug the picker module

```
:lua vim.print(vim.tbl_keys(require('ccc').picker))
```
This will show all available picker functions.

### Check if your plugin configuration loaded

```
:lua print(package.loaded['plugins.ccc-custom'])
```
