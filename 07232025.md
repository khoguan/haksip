## neovide 初步設定

1. neovide 佮 IMK (Imput Method Kit) 有相容性个問題，結束 neovide 个時會有相關个
錯誤訊息，Warp AI 叫我加 --no-fork 佇 alias 內底，無效。目前直接 `2>/dev/null`。

2. `:let g:neovide_cursor_animation_length=0` 關掉 neovide cursor 動畫功能

3. neovide 無真穩，有時會當掉。

4. Raycast 加裝 `run shell command` plugin 了後，透過 Raycast 會使執行
   `/Application/Neovide.app` 嘛會使執行 neovide alias。前者是執行預設个
   `~/.config/nvim` 環境，後者是
   `neovide='NVIM_APPNAME="nvim-as5" neovide --no-fork 2>/dev/null'`。
   前者个字體真細，後者會使得用 Ctrl = 號來放大，用 Ctrl - 號來縮小。
   （neovide 用 ligature font，= 佮 > 做伙會變做一个大箭號 => ）

   補充：
   Ctrl= 佮 Ctrl- 放大／縮小字型个功能是 AstroNvim community neovide 提供个設定。
   參考：
   [AstroNvim Configure Neovide](https://docs.astronvim.com/recipes/neovide/)

5. 原則上用 Astro community neovide 就好，我加幾項仔修改：
   * guifont = "Hack Nerd Font:h20", -- neovide font family & size
   * vim.g.neovide_cursor_animation_length = 0
   * vim.bo.linespace = 6

### Warp AI 提供 neovide cursor 動畫控制選項

#### Cursor Animation Length

-  g:neovide_cursor_animation_length - Controls how long the cursor animation
   takes (in seconds)
-  Default: 0.13
-  Example: let g:neovide_cursor_animation_length=0.05 (faster animation)

#### Cursor Trail Size

-  g:neovide_cursor_trail_size - Controls the length of the cursor trail
-  Default: 0.8
-  Example: let g:neovide_cursor_trail_size=0.5 (shorter trail)

#### Cursor Antialiasing

-  g:neovide_cursor_antialiasing - Enables/disables cursor antialiasing
-  Default: v:true
-  Example: let g:neovide_cursor_antialiasing=v:false

#### Cursor Animation Type

-  g:neovide_cursor_vfx_mode - Sets the cursor visual effect mode
-  Options: "railgun", "torpedo", "pixiedust", "sonicboom", "ripple", "wireframe", ""
-  Default: "" (no effect)
-  Example: let g:neovide_cursor_vfx_mode="railgun"

#### Additional VFX Settings

-  g:neovide_cursor_vfx_opacity - Controls opacity of cursor effects (0.0-1.0)
-  g:neovide_cursor_vfx_particle_lifetime - Particle lifetime for effects
-  g:neovide_cursor_vfx_particle_density - Particle density for effects
-  g:neovide_cursor_vfx_particle_speed - Particle speed for effects

You can add these settings to your init.vim or init.lua file to customize the
cursor behavior. To disable cursor animation entirely, you can set
g:neovide_cursor_animation_length=0.
