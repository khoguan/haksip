## 函數圖形平移問題重想

佇座標平面个函數圖形，左右平移 h、上下平移 k 了後（h 佮 k 攏可能是正是負，注意
著按照右正左負、上正下負个正常標記法），舊座標 (a, b) 自然就變做 (a + h, b + k)，
相對來講，函數著愛減掉 h 佮 k 來維持原式个函數關係：
$(y - k) = a(x - h)^2 + b(x - h) + c$。

圖形平移个題型：

1. 已知舊函式、舊點（可能是舊頂點抑是舊函式圖上个任何點），平移 h, k 了後，
   問新圖个函數。就是頂面所寫，但是舊新函數通常用按呢个形式表示：

   - 舊函數： $y = a(x - h_0)^2 + c$  
   - 新函數： $y = a(x - h_0 - h)^2 + c + k$
   - 或寫做： $y - k &= (x - h_0 - h)^2 + c$

3. 條件同上，毋過換問滿足新圖个點。若是頂點，上簡單，就是 (a + h, b + k)；
   若是舊圖頂任意點，除了 (a + h, b + k) 以外，閣愛考慮對稱點个座標，彼嘛仝款
   滿足新圖。

   - 例一：已知點(p, q) 滿足 $y = -x^2$ 个函數關係。若共圖形向左平移 1 單位，
     向上平移 2 單位得著新圖形，an 啥乜點佇新圖形頂？

     解方：由已知可知 $q = -p^2$，而且新點就是 (p - 1, q + 2)，新函數就是
     $y = -(x + 1)^2 + 2$，閣考慮對稱點
     $x + 1 = \pm p \Rightarrow x = \pm p - 1$。所有符合新函數个新點就是
     $(\pm p - 1, q + 2)$。

   - 例二：(a, b) 滿足舊函數 $y = 2x^2 - 8x + 6$，舊函數圖左平移 3，下平移 5
     了後，求滿足新函數个點。

     解方：左移 3、下移 5 个平移量就是 (-3, -5)，若用標準式，新函數就是

     $(y - (-5)) = 2(x - (-3))^2 - 8(x - (-3)) + 6$  
     $(y + 5) = 2(x + 3)^2 - 8(x + 3) + 6$  
     $y = 2x^2 + 4x - 5$

     通常舊函數先配方了後才調整做新函數，按呢較好做：
     $y = 2(x - 2)^2 - 2 \quad\dots\text{(1)}$

     (a, b) 滿足舊函數： $b = 2(a - 2)^2 - 2 \quad\dots\text{(2)}$

     將(1)式改寫做新函數：

     $y - (-5) = 2(x - (-3) - 2)^2 - 2$  
     $y = 2(x + 1)^2 - 7 \quad\dots\text{(3)}$  
     $y = 2x^2 + 4x - 5$
     
     為著欲檢驗新點座標（公式是 $(a + h, b + k)$ 佇遮就是 $(a - 3,\ b - 5)$），
     採用保持配方狀態个 (3) 式較好剃頭：

     $b - 5 = 2(a - 3 + 1)^2 - 7$  
     $b -5 = 2(a - 2)^2 - 7$  
     $b = 2(a-2)^2 - 2$

     算出來有符合(2)式，新函數佮新座標有對同！總是愛閣加上對稱點，也就是 (3) 式个
     $x + 1$ 其實會使等於 (2) 式个 $\pm(a-2)$，
     平方了後正負號就無差矣。

     $$x + 1 = \pm(a - 2)$  
     $x = a-3, -a+1$

     $(-a+1, b-5)$ 代入 (3)：

     $b-5 = 2(-a+1+1)^2 - 7$  
     $b = 2(a-2)^2 - 2$
     
     仝款符合(2)式。

     結論： $(a-3,\ b-5)$ 佮 $(-a+1,\ b-5)$ 攏有佇新函數圖形頂面。

4. 已知舊函數、舊點，閣有新函數，無講按怎平移，直接問新點座標（可能是頂點抑是
   通用點）。

   - 例一：設點 $(a, b)$ 在圖形 $y = 2x^2$ 上，an 啥乜點一定佇圖形
     $y = 2(x-3)^2 -5$ 上？

     解方： $b = 2a^2$，觀察新函數來判斷平移量，舊是 x，新是 x - 3，表示新點是
     舊點右移 3，新函數才著 -3 來維持函數關係（左加右減）；尾溜 -5 就是下移 5，
     新點就是 $(a+3, b-5)$，但是愛閣考慮對稱點：

     $x - 3 = \pm a$  
     $\therefore\ x = \pm a + 3$

     完整个新點就是 $(\pm a + 3, b - 5)$。

   - 例二：設 $(s,t)$ 佇 $y=3(x-1)^2$ 頂，an 以下佗一點佇 $y=3(x+2)^2-7$ 頂？

     解方：由條件知， $t=3(s-1)^2$。做熟了後，嘛免閣觀察新函數來判斷平移量，
     直接將配方過个新函數化做舊函數个形而且考慮對稱點：

     $x + 2 = \pm (s-1)$  
     $\therefore\ s = s - 3,\ -s - 1$

     所以新點座標有兩款： $(s - 3,\ t - 7),\ (-s - 1, t - 7)$。

## 拋物線函數个本質

配方了後寫做 $y = a(x - b)^2 + c$，即種拋物線个本質會使得講完全由 a 做主：

1. |a| 愈大，開喙就愈大。
2. a > 0，開喙向天，有極小值；a < 0，開喙向地，有極大值。
3. b 佮 c 只是平移量左右佮頂下兩向度，無影響拋物線个形體，干焦影響伊相對於原點
   个位置。
4. 一旦 |a| 決定，b 佮 c 免管，拋物線个形體就一律仝款，無論平移抑是踅180度攏仝。
