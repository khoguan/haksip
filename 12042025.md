## Pascal set vs. array constant

### Set Constant

A set constant uses square brackets [] containing its elements.

Example:

```Pascal
const
  Vowels = ['A', 'E', 'I', 'O', 'U'];  
  Numbers = [1, 2, 3, 4, 5];
```

For enumerated types:

```Pascal
type
  TColor = (Red, Green, Blue, Yellow);

const
  PrimaryColors = [Red, Green, Blue];
```

### Array constant

FreePascal supports array constants using parentheses ().

```Pascal
const
  // Static array constant
  Digits: array[1..5] of Integer = (1, 2, 3, 4, 5);

  // For strings
  Greetings: array[1..3] of string = ('Hello', 'Hi', 'Hey');
```

Record or structured arrays also work:

```Pascal
type
  TIntPair = record
    A, B: Integer;
  end;

const
  Pairs: array[1..3] of TIntPair = (
    (A: 1; B: 2),
    (A: 3; B: 4),
    (A: 5; B: 6)
  );
```

Notes:

✔ Set constants require [].
✔ Array constants require () and must match declared array type.
✔ Sets are limited to ordinal types (integer subranges, char, enum).

Minimal runnable example:

```Pascal
program Demo;

type
  TWeekday = (Mon, Tue, Wed, Thu, Fri, Sat, Sun);

const
  Weekend = [Sat, Sun];                       // set constant
  Days: array[1..3] of TWeekday = (Mon, Wed, Fri);  // array constant

begin
  if Sun in Weekend then
    writeln('Sun is weekend');

  writeln('The first stored day is: ', Ord(Days[1]));
end.
```
