## 修改 plugins/astrocore.lua

```lua
        wrap = true, -- sets vim.opt.wrap
        linebreak = false,
        breakindent = true,
        showbreak = "»",
        -- Show non-breaking spaces as a visible character
        list = true, -- Toggle with <leader>uB
        listchars = { nbsp = "␣", tab = "→ ", trail = "·", extends = "›", precedes = "‹" },
```

`wrap` 佮 `linebreak` 攏只是顯示上个效果，毋是真正拗行，`wrap` 到視窗邊界
才拗行，`linebreak` 會配合 `breakat` 指定个字元可比講標點抑是空格來拗行，
這我用漢字就無適合。雖然 `linebreak` 設做假，但是 `wrap` 為真，按呢
`breakindent` 猶原有作用，會綴拗行進前个縮排來勼縮顯示，加上 `showbreak`
看著就真清楚。另外閣加上 `list` 佮 `listchars` 个效果（上重要个項目是 `tab`
佮 `trail` 非常好用），遮爾肆是个設定，各大 Neovim distros 攏無做甲遮幼路。

## 精進 tokyonight.nvim 个色水設定

原來 `on_highlights = function(highlights, colors) ... end`
兩个參數就是 tokyonight 主題本身提供个色水表格个兩大 key。而且伊有四種 style，
透過 naming abstraction 來予設定个程式碼單純化，毋免 hard code 十六進位个色水碼
入去。
