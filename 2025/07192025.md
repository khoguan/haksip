## 好用个 `:close` command

`:close` - close the current window, 是安全性个指令。會使離開 help windows，
會使離開 Snacks.dashboard。已經將 `:close` 設做 `<leader>c`，閣另將 `:qa` 
設做 `<leader>q`，將 `:qa!` 設做 `<leader>Q`。

1. 無引數進入 nvim 个時，顯示 Snacks.dashboard()，tsit-tsūn 是 1 號視窗

   ```
   :echo winnr()
   1
   ```

2. 選擇一个舊檔開起來，檔案會取代 dashboard 畫面，猶原是 1 號視窗。
3. 等到 `<leader>h` 閣入來 dashboard，tsit-tsūn 就會變做 3 號視窗。
事實是 1 佮 3 號視窗並存，雖然畫面看起來干焦一个視窗。

```
:echo winnr()
3
:windo echo winnr()
1
3
```

4. `<C-w>w` 光標會走去 dashboard 畫面第一逝，用 `:echo winnr()` 會發現是佇 1 號窗，
   閣 `<C-w>w` 光標就走去 dashboard 選單 Find File 遐，閣做 `:echo winnr()`
   就發現是 3 號窗。按呢毋才會使得用 `:close` 關掉 3 號窗，切過去 1 號窗。

5. 進一步試驗，原來 大字橫幅 NEOVIM 个範圍內攏是 1 號窗，對 Find File 迄逝起才是
   3 號窗。3 號窗會使 `:close`，1 號窗袂使！無論光標是佇 1 號抑 3 號，攏會使 `<Esc>`
   離開 dashboard 轉來檔案 buffer。
