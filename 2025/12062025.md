## "be informed by" 个用法

我：Translate this sentence in the preface of the book "Algorithms in C++" to
Chinese:

> On the one hand, our code is informed by experience with implementing
> algorithms in these and numerous other languages (a C version of this book is
> also available, and a Java version will appear soon); on the other hand, some
> of the properties of some of these languages are informed by their designers’
> experience with some of the algorithms and data structures that we consider in
> this book.

Google AI: 这句话的中文翻译是：
一方面，我们的代码融入了使用这些及许多其他语言实现算法的经验（本书的 C 语言版本
也已推出，Java 语言版本即将面世）；另一方面，这些语言的某些特性也得益于其设计者
在使用本书所述及的某些算法和数据结构方面的经验。

我：我對於其中 "be informed by" 這個用法感到困惑，不知如何掌握它的意義。

Google AI: 「be informed by」在這裡的意思並不是「被告知」或「得知」，
而是指「受到影響」、「以⋯⋯為基礎」或「融合了⋯⋯經驗」。它表達的是一種影響或
塑造的來源。

在您提供的句子中，有兩個地方使用了這個片語，我們可以拆開來看：

> our code is **informed by** experience

- 直譯：我們的程式碼受到經驗的影響/啟發。
- 意譯：我們的程式碼融合了（或借鑒了）實現演算法的經驗。
- 強調了「經驗」是撰寫程式碼的依據或基礎。

> properties of some of these languages are **informed by** their designers’
> experience

- 直譯：這些語言的特性受到設計者經驗的影響/塑造。
- 意譯：這些語言的某些特性得益於（或建立在）設計者使用演算法和資料結構的經驗之上。
- 強調了「設計者的經驗」是語言特性形成的原因或基礎。

簡單來說，這個用法強調的是某事物的形成、性質或決策過程，是受到另一事物的知識、
經驗或數據所影響或指導。

掌握這個片語的關鍵在於理解它表達的是間接且深刻的影響，而不僅僅是表面的
「通知」動作。
