## 改 bamboo theme 个正文光度

bamboo 个說明書內底有講伊是特別設計欲減輕目睭个壓力，我感覺佇 MD 顯示起來，
正文个光度猶原傷強。今來改看覓。

家己改袂好勢，叫 Warp AI 來鬥改，主要也是用 TreeSitter node names，伊提可能
有關个 node names 攏共改色，予伊啄著米矣！

```lua
    -- Create darker foreground color for markdown
    local markdown_fg = util.darken(bamboo_colors.fg, 0.15) -- 0~1 lighter to darker

    -- Set up an autocommand to apply darkened text for Markdown files
    vim.api.nvim_create_autocmd({ "FileType" }, {
      pattern = "markdown",
      callback = function()
        -- Apply darkened foreground to various markdown text groups
        vim.api.nvim_set_hl(0, "@text", { fg = markdown_fg })
        vim.api.nvim_set_hl(0, "@text.literal", { fg = markdown_fg })
        vim.api.nvim_set_hl(0, "@none", { fg = markdown_fg })
        vim.api.nvim_set_hl(0, "markdownError", { fg = markdown_fg })
        -- For regular paragraph text that doesn't match any treesitter group
        vim.api.nvim_set_hl(0, "Normal", { fg = markdown_fg, bg = bamboo_colors.bg0 })
      end,
    })
```

Warp AI 說明：

Multiple highlight groups targeted - The autocommand sets darkened foreground for:

- @text - General text content in Treesitter
- @text.literal - Literal text content
- @none - Plain text without semantic meaning
- markdownError - Markdown-specific text
- Normal - The fallback for regular paragraph text

bambo theme 个 util.darken/lighten 佮 tokyonight 仝名稱个函式，效果倒反，數值
愈倚 1 效果愈強，darken 愈暗，lighten 愈光。tokyonight 是以 1 表示原光度，所以
數值愈倚 0 效果愈強。

## 了解 mason-null-ls.nvim, conform.nvim, nvim-lint 个關係

community.pack.lua 內底即三種 plugin 攏有用著，攏設 optional，實際安裝个時
無應該正實三套攏裝。mason-null-ls 較包含，linter 佮 formatter 攏有才調處理；
conform 專門處理 formatter，nvim-lint 專門處理 linter。所以 mason-null-ls 內底
有設置 stylua(a formatter) 佮 selene(a linter)；conform 干焦設置 stylua，
nvim-lint 干焦設置 selene。即三套「管控器」攏有內建一堆有支援个
linters/formatters，通常直接引用一行 code 就會使，伊會替咱匯入相關个設置檔，
咱欲另外改一寡參數也會使。

個別个 linter/formatter 也有家己个 config. file，像 selene 愛用 `selene.toml`
佮 `neovim.yml`；luacheck 用 `.luacheckrc`；stylua 用 .stylua.toml。

我看著 Josen 个 Youtube 影片了後才較有概念。

mason-null-ls 是 null-ls 加上包裝；後來 null-ls 停止維護，閣有 none-ls 出現。
