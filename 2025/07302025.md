## 簡單比較三種顯色內插

- **Colorizer**: color codes, color names 攏會顯色，但是干焦提供背景佮前景兩種
  顯色模式，無 virtual 通選。

- **ccc**: color codes, ANSI escape sequence 兩類會顯色，無處理 color names，
  但是有另提供色緻選擇器，是額外个大功能。

- **NvChad/ui**: color codes, color names 攏會顯色，也有 bg/fg/virtual 三種
  顯色模式通用。但是愛掛`nvchad/base46`做伙安裝，見擺 `:write` 了後，我个
  night-owl 主題个色水就會走爭去。

## 顯色插件 `brenoprata10/nvim-highlight-colors`

伊即套插件，顯色个色彩類型算有tsiâu，fg/bg/virtual 攏有，ANSI color codes,
named colors 攏有支援。但是毋是凊彩扑一篇文字檔，內檔个英文色彩單字伊攏會顯色，
主要是佇 CSS 檔內，抑是一寡特殊語境才會顯色，按呢也是合理啦。

我叫 Warp AI 共我創一个 CSS 色水測試檔 `color-test.css`，袂䆀，各種類型个色水攏有。
不而過，以下个格式伊就無掠著：
```

    background: linear-gradient(45deg, 
        firebrick, 
        goldenrod, 
        olivedrab, 
        steelblue, 
        mediumpurple
    );
}
```
