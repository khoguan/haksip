## Bash Scripting

需求：欲將一堆檔名xxxxxxxx-ddd.pdf 改做 ddd.pdf
所在：/Volumes/Extreme/com_ebooks/Pascal/Pascal Plus Data Structures, Algorithms, and Advanced Programming (1995)/wip

初版 script:

```Bash
#!/bin/bash

for F in *.pdf
do
  NF=`echo ${F} | sed 's/^.........//'`
  mv -v ${F} ${NF}
done
```

透過 nvim 內底 shellcheck linter 个輔助佮 Warp AI
个指導，學著袂少撇步。修訂版 script：

```Bash
#!/bin/bash

# 假使 *.pdf 無 match 著檔案，預設會變做無變个 '*.pdf' 字串，
# shopt 設定 nullglob 就會共他設做 null，才袂進入 for loop
shopt -s nullglob
for F in *.pdf
do
  # 用 ".." 包起來，才讀會著完整个檔名字串，袂因為檔名有空格來重耽。
  # substring operator 對尾後算起來第7个字元開始取到上後尾
  # 注意愛空一格，毋通寫做 ${F:-7}
  # cf. Bash parameter expansion
  NF="${F: -7}"
  # 用 echo 先試，無問題才真正執行 mv
  # echo "mv -v $F $NF"
  mv -v "$F" "$NF"
done

# 執行
# diff <(./ren2.sh | awk '{print $4}' | sort | sed 's/.pdf//') <(seq 108 267)
# 發現欠 p.188 佮 p.266
```
