# 2025-09-02 學習日誌

- [ ] Comment.nvim in AstroNvim is disabled by default for Neovim 0.11+

## Comment.nvim in AstroNvim

今仔日是先父往生第十九年个忌辰。早起起來，閣有一屑仔時間，安裝 Comment.nvim，
發現 `:Lazy` 顯示 disabled。緊叫 Warp AI 共我查，原來 AstroNvim 看著 Neovim 0.11
以上个版本，會將 Comment.nvim 禁起來。

這我進前就有注意著，0.11+ 个 Neovim 有提供一部份 Comment.nvim 个功能佮 keymaps 矣，
總是無完整，Comment.nvim 有 block comment keymaps `gbc` 佮 `gb{motion}` 這目前
Neovim 猶未內建。

Neovim 有一直咧將一寡 plugins 个好功能整合入來，不而過，伊會較保守、謹慎，所以往往
採用一部份爾爾。
