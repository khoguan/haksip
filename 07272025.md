## fzf 解破

fzf 本質上只是對標準輸入讀入一个列表，閣將列表輸出去標準輸出。但是因為伊有
種種奇妙个整合性功能，所以才大受歡迎。

### fzf 環境變數

```
FZF_CTRL_T_COMMAND='fd --type f --strip-cwd-prefix --hidden --follow --exclude .git'
FZF_CTRL_T_OPTS='--preview "bat --color=always --line-range :500 {}"'
```
前者提供 fzf 个輸入內容，通常就是專案首目錄以下所有咱可能欲選用个檔案列表，
了後 fzf 个輸出，左欄是簡單个列表，右欄是根據後者所設，利用 bat 做 preview。

```
FZF_ALT_C_COMMAND='fd --type d . --color=never --hidden --exclude .git'
FZF_ALT_C_OPTS='--preview "tree -C {} | head -200"'
```
以上兩个變數仝原理，毋過是針對目錄个操作。

### fzf 上重要个四種使用情境（攏是佇 shell 當中）

1. 針對檔案：<Ctrl-T> 叫出 CWD 下底所有个檔案予咱選
   -  執行个是 $FZF_CTRL_T_COMMAND 變數所設个字串
   -  例：`~/.config/nvim/> nvim <C-t>`
   -  伊會使予儂用 <tab> 複選幾若个檔案，方便 nvim 做一睏編輯專案目錄下底
      分散佇無仝子目錄个檔案，蓋好用！

2. 針對目錄：<Alt-C> 叫出 CWD 下底所有个目錄予咱選
   - 執行个是 $FZF_ALT_C_COMMAND 變數所設个字串
   - 我个Mac環境所對應个 Alt key 是 Esc。我个 Alt key 其實是 Opt Key。

3. nvim foo**<tab>

4. cd bar**<tab>
