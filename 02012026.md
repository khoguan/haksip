## 平方數列个通項

若是以項次 k 來論，每一項當然是 k^2。但是佇 Raku 語言个 Sequence type 來講，
咱愛用前一項个值做基礎來指定本項个值：

```Raku
# 底數固定，變指數，有效
my $seq1 := 1, 2, 4, 8 ... *;
for $seq1[^20] { .say };

# 指數固定，變底數，無效
my $seq2 := 1**2, 2**2, 3**2, 4**2, 5**2 ... *;
for $seq2[^20] { .say };

# 用指定个，會使得！
my $seq3 := 1, 4, 9, 16, 25, { $_ + 2 * sqrt($_) + 1 } ... *;
for $seq3[^20] { .say };
```

前一項个值假使是 a，本項个值就是 $a + 2\sqrt{a} + 1$，這是按怎算出來个？

用正方磚相疊來想，上頂懸棧是第一項 1² 塊磚，落來第二棧是 2² 塊磚，落來第三棧 3²
塊磚，而且排法是予頂一棧集中佇仝一角，可比講西北角，按呢俯視看會著个第四棧个磚仔
干焦有南勢橫排三塊磚加上東勢直排三塊磚（= 2 * √a = 2 * √9 = 2 * 3），閣加上
東南角一塊磚爾爾。第四棧其他个磚仔，因為攏予頂懸棧个磚仔疊咧，自然就看袂著。
按呢第四棧磚仔个總數就是予疊咧看袂著个頂一棧个總數 a + 2 * √a + 1。

## MathJax 指令試驗

cf.
1. https://treeofmath.github.io/tex-commands-in-mathjax/TeXSyntax.htm
2. 

`Typora` 佮 `markdown-preview`，甚至是 `render-markdown` 對 MathJax 指令个支援
情形有出入。以下是各種个試驗：

$$
\begin{align}
   v + w & = 0  &&\text{Given} \tag 1\\
   -w & = -w + 0 && \text{additive identity} \tag 2\\
   -w + 0 & = -w + (v + w) && \text{equations $(1)$ and $(2)$}
\end{align}
$$

以上第三行个數式照講無應該有編號，但是 preview 出來有！

以下是 `\begin{align} ... \end{align}` 个效果：

$$
\begin{align}
a   &= bbb&bbb &=   c&c   &= d \\
aaa &=  bb&bb  &= ccc&ccc &= ddd
\end{align}
$$

以下是 `\begin{alignat} ... \end{alignat}` 个效果，欄位內容較束結：

$$
\begin{alignat}{3}
a   &= bbb&bbb &=   c&c   &= d \\
aaa &=  bb&bb  &= ccc&ccc &= ddd
\end{alignat}
$$
