## 已知兩點，求直線方程式 Ax + Bx + C = 0 个 A, B, C

已知兩點(1,3), (-2, -1)，求直線方程式 Ax + Bx + C = 0 个 A, B, C。

### 直接套兩點入去解聯立方程式

咱若戇戇仔將兩點代入 Ax + By + C = 0，就會發現愛用兩條方程式來解三个未知數。
這顯示佇本質上，A, B, C 毋是定值，只是比例个關係爾爾。

$$
\begin{cases}
A + 3B + C &= 0 & \qquad (1) \\
-2A - B + C &= 0 & \qquad (2) \\
\end{cases}
$$

去 C 求 A, B 比例：
$$
(1)-(2)\Rightarrow 3A + 4B = 0 \\
A = \frac{-4B}{3} \qquad(3)
$$

去 A 求 B, C 比例：
$$
(1)\times 2 + (2) \Rightarrow 5B + 3C = 0 \\
C = \frac{-5B}{3}\qquad(4)
$$

按呢就簡單得著 A, B, C 个比例值：
$$
A:B:C = \frac{-4}{3} : 1 : \frac{-5}{3} = 4 : -3: 5 \\[1ex]
\therefore \text{直線方程式是 } 4x - 3y + 5 = 0
$$
驗證代入已知兩點
$$
(1,3)\Rightarrow 4\cdot 1 - 3\cdot 3 + 5 = 0 \\
(-2,-1)\Rightarrow 4\cdot(-2) -3\cdot(-1) + 5 = 0
$$
正確！

### 用兩點式公式

$$
\frac{y - 3}{x - 1} = \frac{3-(-1)}{1-(-2)} = \frac{4}{3} \\
3y - 9 = 4x - 4 \\[1ex]
\therefore\text{直線方程式是 }4x - 3y + 5 = 0
$$

## 用斜截式較好理解直線方程式个本質

上簡單个直線方程式是垂直線 $x = \alpha$ 佮水平線 $y = \beta$。其中 $x=0$
就是 y 軸，$y=0$ 就是 x 軸。

閣來就是通過原點个斜線。某一條斜線頂面任何一點个 x, y 一定攏有
$m = \dfrac{y}{x}$，m 是定值个性質，這會使得用相似直角三角形來證明。即个 m 就是
斜率。寫做函式就是 $y = mx$。

閣來就是上下平移 k（k 可正可負），按呢就是 $y = mx + k$，整理做 $mx-y+k=0$。

既然斜截式這麼好用，為什麼課本還要教那個麻煩的「一般式」？
主要是為了補齊斜截式的兩個致命缺點：

1. 垂直線的「無力感」  
斜截式有一個唯一的剋星，就是垂直線（例如 x = 5）。
   - 垂直線的斜率是「不存在」的（分母為 0）。
   - 你無法在這個公式裡找到一個數字填入，讓它變成一條垂直線。
   - 這意味著，如果你只用斜截式，你沒辦法用同一個公式處理全世界所有的直線。
2. 數學上的「對等性」  
在進階數學中，我們喜歡「對稱」和「統一」。
   - 在 $Ax + By + C = 0$ 中，x 和 y 的地位是平等的。
   - 它不只包含垂直線，還能很方便地推導出一些進階公式（例如：點到直線的距離
     公式）。如果用斜截式來寫距離公式，公式會長得非常醜陋且複雜。

潘按：點(h, k)到直線 Ax + By + C = 0 个距離公式 $\dfrac{|Ah + Bk + C|}{\sqrt{A^2+B^2}}$，確實需要用直線一般式較好表示。
